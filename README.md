# Mayoof Lawyers Website

Official website package for Mayoof Lawyers, prepared for GitHub and Vercel.

## Deploy with GitHub and Vercel

1. Extract this ZIP.
2. Create a new GitHub repository and upload all extracted files.
3. In Vercel, choose **Add New Project** and import the GitHub repository.
4. Keep the detected framework as **Next.js** and click **Deploy**.

No environment variables or database setup are required.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
